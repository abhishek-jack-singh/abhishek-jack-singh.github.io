Projects

Understanding and predicting chronic absenteeism among secondary grade students.

This project aims to predict student absenteeism and identify both predictive and causal factors of absenteeism. Thus far demographics, student behavior, health conditions, student involvement in co-curricular activities, and household information have been used in predictive models of absenteeism. These models reveal that absenteeism is related to demographics, student behavior, health conditions and household variables. Currently, we are trying to improve the predictive model’s recall for chronically absent students by re-engineering the existing features and adding new features.

Revolutionizing Dense Medium Cyclone Performance using Convolution Neural Network

This project pioneers precise operating parameter estimation for dense medium cyclones. A dense medium cyclone is a device that uses centrifugal force to separate materials according to specific gravity in the medium. Our focus lies on outlet discharge characterization—spray or rope. We employed a convolutional neural network to classify discharge types with a remarkable 97.2% accuracy; trained on triaxial accelerometer vibrational data collected via experiments. Presently, our efforts are dedicated to an innovative regression model predicting medium concentration.

Predicting Gross Calorific Value and Location of Coal using Neural Networks

In partnership with IIT-Kharagpur's PhD candidate, Majid Hasan Tyeb, this project focuses on employing Multivariate Linear Regression and Artificial Neural Network based approaches to develop Gross Calorific Value (GCV) prediction models using proximate parameters— Moisture, Volatile Matter and Ash content, of coal. Both MLR and the ANN model could predict the GCV of coal with an R2 of 0.934 and 0.962, respectively. We also developed a fully connected neural network comprising of two hidden layers in a MLP architecture to predict the location of the coal sample using the proximate parameters and GCV.
